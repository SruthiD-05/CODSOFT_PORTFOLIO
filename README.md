# Personal Portfolio Website

This project is a simple personal portfolio website created using **HTML and CSS** as part of the **CodSoft Web Development Internship (Level 1 Task 1)**.

## Overview
The portfolio website showcases my personal information, skills, projects, and contact details. It is designed to provide a brief introduction about me and highlight the work I have done.

## Features
- Navigation menu
- About Me section
- Skills section
- Education section
- Projects section
- Resume download button
- Contact information
- Clean and responsive layout

## Technologies Used
- HTML
- CSS

## Purpose
The purpose of this project is to practice front-end web development by creating a structured and visually appealing personal portfolio.
